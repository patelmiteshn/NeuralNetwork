CURRENT NEURAL NETWORK (NN) ARCHITECTURE\
\pard\tx220\tx720\pardeftab720\li720\fi-720
\ls2\ilvl0\cf0 {\listtext	\'95	}Current code can only handle one hidden layer\
{\listtext	\'95	}Current setup required the data be separated training-validation-testing before hand\
{\listtext	\'95	}The NN architecture is limited to a feed forward neural network with back propagation.\
{\listtext	\'95	}The NN uses bipolar logistic function as the activation function.\

  INPUT NODES:\
\pard\tx940\tx1440\pardeftab720\li1440\fi-1440
\ls3\ilvl1\cf0 {\listtext	\uc0\u8259 	}The number of input nodes of the NN is calculated from the input of the training data file. \
{\listtext	\uc0\u8259 	}It used the length of the rows in the data file to determine the number of input nodes\

\cf0   HIDDEN NODES:\
\pard\tx940\tx1440\pardeftab720\li1440\fi-1440
\ls4\ilvl1\cf0 {\listtext	\uc0\u8259 	}The default number of hidden nodes in the single layer are calculated using the formula as under:\
{\listtext	\uc0\u8259 	}(hiddenNodes_ = ceil((pow(outputNodes_,2.0) + outputNodes_+ 2)/2)+1. \
{\listtext	\uc0\u8259 	}The number of hidden nodes can also be specified through the command line. \

\pard\tx940\tx1440\pardeftab720\li1440\fi-1440
